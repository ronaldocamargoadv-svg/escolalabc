# Documentacao do Projeto Escola LaBC

Este diretorio centraliza os documentos de produto, gestao, arquitetura,
qualidade, seguranca e operacao da plataforma Escola LaBC.

## Estrutura

- `produto/`: documentos de visao, proposta, especificacao e backlog.
- `gestao/`: squad, RACI, plano de execucao, decisoes e sprint.
- `tecnico/`: arquitetura, API, modelo de dados e referencias de implementacao.
- `qualidade/`: plano de testes, homologacao e validacao do MVP.
- `linguagens-e-stack.md`: tecnologias, linguagens e responsabilidades.
- `arquitetura-funcional.md`: mapa funcional da solucao.
- `seguranca-lgpd.md`: diretrizes de seguranca e protecao de dados.
- `runbook-operacional.md`: comandos, rotinas e verificacoes locais.
